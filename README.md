# SigForge

**Design a professional email signature with live preview. Copy the HTML into Gmail, Outlook, or Apple Mail.**

Three templates, full customization, email-safe HTML output. No account needed.

## Templates

| Template | Best For |
|----------|----------|
| Minimal | Clean, text-only signatures |
| With Logo | Professional with company logo |
| With Banner | CTA banner for promotions/booking links |

## Fields

- Full name, job title, pronouns
- Email, phone, company, website
- Brand color (hex picker)
- Logo URL + width
- LinkedIn, Twitter/X links
- CTA banner image + link (banner template)

## Output

Table-based HTML with inline CSS. Works in:
- Gmail (paste into Settings > Signature)
- Outlook (paste into signature editor)
- Apple Mail (paste into Preferences > Signatures)

Two copy modes:
- **Copy HTML** — raw HTML source code
- **Copy as Rich Text** — formatted paste for visual editors
